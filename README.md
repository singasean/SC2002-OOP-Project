# SC2002-OOP-Project
A Java console application for managing student internships. Students can browse and apply for internships, companies can post positions, and staff can oversee the entire process.

## Features
- **Students**: Browse, filter, and apply for internships
- **Company Representatives**: Post internships and review applications
- **Career Center Staff**: Approve registrations, internships, and generate reports

## Prerequisites
- Java JDK 17 or higher
- Java IDE

## How to Run
1. **Clone the repository**
2. **Compile the code**
3. **Run the application from "Main.java"**
4. **Troubleshooting CSV Files**
   
If the program is unable to read the CSV files, redownload the 3 CSV files:
- `sample_company_representative_list.csv`
- `sample_staff_list.csv`
- `sample_student_list.csv`

Place them in the project root folder and rerun the code.
